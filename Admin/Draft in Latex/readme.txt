This package is based on the ESCAPE 25 LaTeX template by Heinz A Preisig,
Chem Eng, NTNU, Trondheim, Norway, heinz.preisig@chemeng.ntnu.no

The ESCAPE 25 template can be found here:
http://www.pse2015escape25.dk/files/ESCAPE25.zip

This package contains

* readme.txt      (this file)
* escape31.sty    (ESCAPE 31 style file)
* left_eq.sty     (equation style file)
* elsart-harv.bst (bibliography style)
* example.tex     (sample TeX file)
* example.bib     (sample BibTeX file)
* example.pdf     (sample pdf file generated from corresponding TeX file)
* F_drop.eps      (sample eps graphics)
